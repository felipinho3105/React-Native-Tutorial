# tópicos importantes

1 - Qual a linguagem de programação utilizada na disciplina?
    JavaScript
2 - O que é node?
    Onde o JavaScript roda
3 - O que é npm?
    Node Package Manager - o que gerencia 
4 - O que é um FrameWork e qual a diferença para uma biblioteca?
    FrameWork é um conjunto de recursos programados para resolver problemas, qu ejá tem uma opinião, uma "estrutura" para ser seguida, sendo ela mais rigida ou não. Já a biblioteca é um conjunto de recursos programados para resolver problemas, mas sendo totalmente flexível e sem opinião ou "estrutura" para ser seguida.
5 - O que é react?
    Uma biblioteca 
6 - O que é react native?
    Uma biblioteca usada para o mobile, que substitui o React DOM
7 - O que é expo?
    Framework utilizado para desenvolver aplicações mobile
8 - Como estilizar um componente native?
    importar o StyleSheet e usar a função create para estilizar
9 - O estilo do React Native e do CSS é igual? Tem todas as propriedades? 
    Não, o native é muito inspirado no CSS, então é muito parecido, mas algumas funções, que não faria sentido ter no desenvolvimento mobile, não existem no native
